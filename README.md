## Installation

### Option 1 (Recommended): Build from Source

```bash
# Clone the repository
git clone https://github.com/ashworth3/BatteryClock.git
cd BatteryClock

# Run the install script
./install.sh

# Open the built widget:
open .build/release/BatteryClock.pock
```
